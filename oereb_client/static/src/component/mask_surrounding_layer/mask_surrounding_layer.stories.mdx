# Mask Surrounding Layer

This component binds the visibility of the mask surrounding layer to the corresponding Redux slice.

```jsx
<OerebMaskSurroundingLayer maskSurroundingLayer={maskSurroundingLayer} />
```

### Parameters

| Param | Type | Description |
| --- | --- | --- |
| maskSurroundingLayer | <code>ol.layer.Tile</code> | The layer instance showing the maks surrounding map. |
