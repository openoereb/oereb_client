# Real Estate Layer

This component binds the visibility of the real estate layer to the corresponding Redux slice.

```jsx
<OerebRealEstateLayer map={map} realEstateLayer={realEstateLayer} />
```

### Parameters

| Param | Type | Description |
| --- | --- | --- |
| map | <code>ol.Map</code> | The map instance. |
| realEstateLayer | <code>ol.layer.Vector</code> | The layer instance showing the current real estate. |
