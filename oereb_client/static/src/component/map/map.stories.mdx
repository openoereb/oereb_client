# Map

A wrapper component containing the map element for OpenLayers and layer and query components:

- `OerebMapQuery`
- `OerebRealEstateLayer`
- `OerebTopicLayer`
- `OerebAvailabilityLayer`

```jsx
<OerebMap />
```
