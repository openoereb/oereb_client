# Topic Layers

This component binds the visibility of the topic layers to the corresponding Redux slice.

```jsx
<OerebTopicLayer topicLayers={topicLayers} />
```

### Parameters

| Param | Type | Description |
| --- | --- | --- |
| topicLayers | <code>ol.layer.Group</code> | The layer group containing the topic layers. |
| tiled | <code>bool</code> | Set true, to use tiled WMS requests. |
