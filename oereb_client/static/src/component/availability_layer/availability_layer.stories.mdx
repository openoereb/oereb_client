# Availability Layer

This component binds the visibility of the availability layer to the corresponding Redux slice.

```jsx
<OerebAvailabilityLayer availabilityLayer={availabilityLayer} />
```

### Parameters

| Param | Type | Description |
| --- | --- | --- |
| availabilityLayer | <code>ol.layer.Tile</code> | The layer instance showing the availability map. |
