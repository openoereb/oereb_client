# addIfNotContains

Adds an element to an array only if the array does not already contain the element.

```js
addIfNotContains(item, target);
```

### Parameters

| Param | Type | Description |
| --- | --- | --- |
| item | <code>*</code> | The element to add. |
| target | <code>Array</code> | The target array. |
