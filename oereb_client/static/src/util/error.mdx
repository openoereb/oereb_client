# NoDataError

Custom error class for handling no-data exceptions on `getegrid` requests.

```js
new NoDataError(message);
```

### Parameters

| Param | Type | Description |
| --- | --- | --- |
| message | <code>String</code> | The error message. |

### Returns

<code>NoDataError</code> The error instance.

# TooManyRequestsError

Custom error class for handling HTTP 429 (Too Many Requests) exceptions.

```js
new TooManyRequestsError(message);
```

### Parameters

| Param | Type | Description |
| --- | --- | --- |
| message | <code>String</code> | The error message. |

### Returns

<code>TooManyRequestsError</code> The error instance.
