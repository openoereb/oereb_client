# addDocumentIfNotContained

Adds the document to the specified array if it not already existing in the array.

```js
addDocumentIfNotContained(document, target);
```

### Parameters

| Param | Type | Description |
| --- | --- | --- |
| document | <code>Object</code> | The document to be added. |
| target | <code>Array</code> | The target array. |

# addDocumentsIfNotContained

Accepts an array of documents and a target array and adds only the missing documents.

```js
addDocumentsIfNotContained(document, target);
```

### Parameters

| Param | Type | Description |
| --- | --- | --- |
| documents | <code>Array</code> | The list of documents to be added. |
| target | <code>Array</code> | The target array. |
