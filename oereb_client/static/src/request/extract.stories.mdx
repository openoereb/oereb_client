# queryExtractById

Queries the extract for the specified E-GRID and language.

```js
queryExtractById(applicationUrl, egrid, timeout, language);
```

### Parameters

| Param | Type | Description |
| --- | --- | --- |
| applicationUrl | <code>String</code> | The application URL. |
| egrid | <code>String</code> | The E-GRID to query the extract for. |
| timeout | <code>Integer</code> | The specified timeout in seconds. |
| language | <code>String</code> | (OPTIONAL) The requested language. |

### Returns

<code>Promise</code> A promise receiving the extract data.


# queryStaticExtractById

Queries the static extract (PDF) for the specified E-GRID and language.

```js
queryStaticExtractById(applicationUrl, egrid, language);
```

### Parameters

| Param | Type | Description |
| --- | --- | --- |
| applicationUrl | <code>String</code> | The application URL. |
| egrid | <code>String</code> | The E-GRID to query the static extract for. |
| timeout | <code>Integer</code> | The specified timeout in seconds. |
| language | <code>String</code> | (OPTIONAL) The requested language. |

### Returns

<code>Promise</code> A promise receiving the static extract document.

### Parameters

| Param | Type | Description |
| --- | --- | --- |
| code | <code>String</code> | The topic code. |

### Returns

<code>String</code> The sanitized topic code.


# getParentTheme

A helper function to determine the parent theme for a specific subtheme
out of the list of concerned themes.

```js
getParentTheme(theme, concernedThemes);
```

### Parameters

| Param | Type | Description |
| --- | --- | --- |
| theme | <code>Object</code> | The subtheme object. |
| concernedThemes | <code>Array</code> | The list of concerned themes. |

### Returns

<code>Object</code> The parent theme.


# groupRestrictionsByTopic

A helper function grouping the available restrictions by topic code for easier access.
The concerned themes are needed to find the parent theme for subthemes.

```js
groupRestrictionsByTopic(restrictions, concernedThemes);
```

### Parameters

| Param | Type | Description |
| --- | --- | --- |
| restrictions | <code>Array</code> | The list of restrictions. |
| concernedThemes | <code>Array</code> | The list of concerned themes. |

### Returns

<code>Object</code> The grouped restrictions.
