Filter components in oereb

Name Description
format

Searches for "{}" marks in the input text and replaces them with the provided values.
mark

Marks all occurrences of the specified substring.
multilingualText

Extracts the specified language or default language from a multilingual array.
replace

Replaces all occurrences of the specified substring with the defined replacement.
searchInformation

Search for the specified input in glossary or exclusion of liability records.
searchResult

Extracts the information out of the search result string provided by search API.
orderGlossary

Sorts the glossary elements respecting the specified language.
sortLegend

Sorts the legend elements respecting their sub theme, area and length.
