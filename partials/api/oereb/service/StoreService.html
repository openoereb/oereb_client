StoreService

  1. - service in module oereb

Angular service to handle the local browser storage.

Usage

StoreService($rootScope, oerebEventSymbolZoomEnabled, oerebLocalStoragePrefix);

Arguments

Param Type Details
$rootScope angular.Scope

The application's root scope.
oerebEventSymbolZoomEnabled string

Symbol zoom status event name.
oerebLocalStoragePrefix string

Prefix for the localStorage properties.
