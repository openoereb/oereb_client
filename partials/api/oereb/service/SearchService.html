SearchService

  1. - service in module oereb

Angular service to handle a search term passed to GeoViewBL search api.

Usage

SearchService($http, $q, searchServiceConfig, wfsFilterServiceUrl);

Arguments

Param Type Details
$http angular.$http

Angular service for HTTP requests.
$q angular.$q

JSON-encoded search configuration.
wfsFilterServiceUrl string

The WFS url.
