oerebThemeWithoutData

  1. - directive in module oereb

Collapsible element containing the themes without data.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <oereb-theme-without-data
  toggled-group="string">
...
</oereb-theme-without-data>

Arguments

Param Type Details
toggledGroup string

The currently toggled group.
