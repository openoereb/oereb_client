oerebLegalDocuments

  1. - directive in module oereb

Element containing the legal documents for a specified topic.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <oereb-legal-documents
  theme-code="string">
...
</oereb-legal-documents>

Arguments

Param Type Details
themeCode string

The code of the theme to show the legal documents for.
