multilingualText

  1. - filter in module oereb

Extracts the specified language or default language from a multilingual array.

  • If no language is specified, the default language will be used.
  • If no language matches, the first element will be returned

Usage

In HTML Template Binding

{{ multilingualText_expression | multilingualText}}

In JavaScript

$filter('multilingualText')(language)

Arguments

Param Type Details
language string

The language to extract.
