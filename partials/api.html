API reference

The OeREB Client application is built using the following frameworks and libraries:

  • AngularJS (1.6.6)
  • Twitter bootstrap (3.3.7)
  • Font awesome (4.7.0)
  • jQuery (1.9.1)
  • Openlayers (4.2.0)
  • proj4 (2.4.4)

The application code is compiled and minified with the Google Closure Compiler.
